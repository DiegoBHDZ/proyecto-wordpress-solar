# WordPress Custom Theme - Blog

Custom WordPress theme with complete blog system.

## Requirements

- Docker
- Docker Compose

## Installation

```bash
# Clone repository
git clone <repository-url>
cd mi_blog_prueba

# Run setup
./setup.sh
```

## Access

**URL:** http://localhost:8080

**Credentials:**
- Username: `admin`
- Password: `admin`

## What's Included

- Custom theme "Tema Personal"
- 3 demo blog posts
- Responsive design
- Widget system
- Configured menus

## Commands

```bash
# View logs
docker-compose logs -f

# Stop containers
docker-compose down

# Restart
docker-compose restart
```
