== Description ==

Tema Solar Energy es un tema moderno de WordPress diseñado para empresas de energía solar y renovable. Incluye secciones personalizadas para servicios, galería de proyectos, equipo, testimonios y formulario de contacto.

Características:
* Diseño responsive (mobile-first)
* Secciones modulares reutilizables
* Carrusel de hero con Bootstrap
* Galería de proyectos con Lightbox
* Formulario de contacto
* Soporte para imágenes destacadas
* Compatible con widgets
* Optimizado para SEO

== Installation ==

1. Sube la carpeta 'tema-solar' a '/wp-content/themes/'
2. Activa el tema desde el panel de WordPress en Apariencia > Temas
3. Crea una página nueva y asígnale el template "Home Solar"
4. Establece esa página como página de inicio en Ajustes > Lectura

== Frequently Asked Questions ==

= ¿Cómo configuro la página de inicio? =

Crea una nueva página, asígnale el template "Home Solar" desde el editor, y luego ve a Ajustes > Lectura y selecciónala como página de inicio estática.

= ¿El tema es responsive? =

Sí, el tema está completamente optimizado para dispositivos móviles, tablets y escritorio.

= ¿Puedo personalizar los colores? =

Sí, el tema incluye un archivo theme.json con la paleta de colores configurada que puedes modificar.
