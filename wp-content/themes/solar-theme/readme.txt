== Description ==

Solar Energy is a modern, clean WordPress block theme designed for solar energy companies, renewable energy businesses, and environmental organizations.

The theme features:
* Full Site Editing (FSE) support
* Stunning full-screen hero carousel
* Services/Products showcase grid
* Project photo gallery with Owl Carousel
* Team members section with alternating layout
* FAQs section
* Client testimonials carousel
* Contact form with validation
* Fully responsive design
* Bootstrap 4 framework integration
* Custom color palette optimized for solar energy branding
* Multiple page templates
* Block patterns for easy content creation

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Click 'Upload Theme' and select the theme's ZIP file. Click 'Install Now'.
3. Click 'Activate' to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support Full Site Editing? =

Yes! Solar Energy is a block theme that fully supports WordPress Full Site Editing (FSE). You can customize every aspect of your site using the Site Editor.

= Can I customize the colors? =

Yes! The theme includes a custom color palette that you can modify through the Site Editor or theme.json file.

= Does the theme include a contact form? =

Yes! The theme includes a styled contact form with validation. The form can send emails through WordPress's wp_mail function.

= Is the theme responsive? =

Absolutely! The theme is fully responsive and looks great on all devices.

= What libraries does the theme use? =

The theme integrates several popular libraries:
* Bootstrap 4.4.1
* Owl Carousel 2
* Lightbox 2
* Isotope
* Font Awesome 5.10.0
